# ritual
The ritual!

## Setup
Install python 3
https://www.python.org/downloads/

Install pipenv
```shell
pip install --user pipenv
```

Install dependencies:
```sh
pipenv install
```

## Run
Be sure to activate the shell environment.
```
pipenv shell
```

Run any of the rituals
```
python ritual_<sacrifice>.py
```

## Build
```sh
pipenv run deploy.bat
```
